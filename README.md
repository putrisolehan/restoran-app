# Kafe Barokah Abadi - Restaurant Management System

This project is a web-based restaurant management system developed as a group capstone project. The application is designed to help manage menu display, table reservations, and customer orders in an easy and user-friendly way.

## Features

* Interactive homepage with image slider and cafe profile (About section)
* Display of contact information (address, phone number, Instagram)
* Online service overview including ordering and payment information
* Menu catalog with detailed view of each item and pricing
* Table reservation and menu booking system with user input form
* Automatic invoice generation after booking
* Payment instructions via bank transfer and e-wallet 
* Payment verification feature using payment code
* Customer feedback (criticism & suggestions) form

## Tech Stack

* PHP (CodeIgniter)
* MySQL
* HTML, CSS, Bootstrap
